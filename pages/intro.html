Project Proposal

Title: MoodWave

Client: College students and young adults with anxiety/high stress

Objective: To develop an web application that will allow the users to interact and play with different sounds and colors (displayed as “wavelengths”) using a cursor(mouse/trackpad) or hand and arm gestures (LeapMotion). We want to use the Arduino pulse sensor to measure a person’s pulse and breathing rate to see how they would react to the different sounds and colors. With the feedback provided from the Arduino sensor, users can see how the application is helping them manage their anxiety. We will research what sounds/colors usually relax people and suggest users try certain yogic gestures to help relieve anxiety. https://www.sparkfun.com/products/11574 http://jythonmusic.org/index.html

Specifications: Runs on OS X, Windows, and the web. This is an interactive web application that will record a person’s pulse through the Arduino Pulse Sensor, and display different frequencies of sounds and colors, and see how the user reacts. There will be a prompt for the user to use different yogic gestures to calm them down, but the user may use different gestures to play with the different frequencies and color displayed on the screen.

Target Audience: This is primarily for high school/college students and young adults, who experience mild to moderate anxiety and stress. This application will explore how we can use a combination of different methods (sound and color) to help relax a person.
