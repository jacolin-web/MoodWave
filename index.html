Abstract: This project will measure the effect sound and color have on an individual’s mood. The Mindwave EEG headset will be used to measure brain activity of the user. The user will be presented with various colors and sounds/music. The user will be able to interact with the screen while his/her brain activity is monitored. This project will assist a user in getting to a stable/less anxious mood. The results of multiple users will be recorded and stored together, providing helpful data in determining the effect of sound and color on an individual’s mood as a whole.
